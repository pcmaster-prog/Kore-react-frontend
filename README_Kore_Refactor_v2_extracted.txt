interface PageHeaderProps {

}
• Reemplaza el hero oscuro en TODAS las vistas
• En desktop: título grande + acciones alineadas a la derecha
• En móvil (< 1024px): título compacto + acciones en dropdown
1.2 <EmptyState>
interface EmptyStateProps {
level?: 1 | 2 | 3; // Icono | SVG | Lottie
icon?: string; // Nombre del icono Lucide
title: string;
description?: string;
action?: { label: string; onClick: () => void; variant?: string };
variant?: 'celebration' | 'neutral' | 'action'; // celebration = positivo
}
Generated by Kimi.ai
• Nivel 1 (default): Icono Lucide 64px + título + descripción
• Nivel 2: SVG ilustrativo 120px (aplicar en Fase 2)
• Nivel 3: Lottie animado 200px (post-MVP)
• Variantes:
– celebration: Para “Hoy estás libre”, “Todo bajo control” (colores cálidos)
– neutral: Para “No hay datos todavía” (colores grises)
– action: Para “Crea tu primera tarea” (colores primarios + CTA)
1.3 <KpiCard>
interface KpiCardProps {
label: string;
value: number;
icon?: string;
color?: 'blue' | 'yellow' | 'red' | 'green' | 'purple';
hideIfZero?: boolean; // default: true
forceShow?: boolean; // default: false — para casos donde 0 es informativo
trend?: { value: number; direction: 'up' | 'down' }; // opcional
}
• Si hideIfZero=true y value=0 →no renderiza (return null)
• Si forceShow=true →siempre renderiza, incluso con 0
• Colores predefinidos con tokens de Tailwind
